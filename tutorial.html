NERDLAND FESTIVAL - CODE A SOUNDBOARD

We gaan een interactief soundboard maken!
We kunnen er korte grappige samples kunnen afspelen, èn er ook opnemen, om die vervolgens toe te voegen.

Daarvoor hebben we drie codetalen nodig

  • HTML
  • CSS
  • JavaScript

Geen paniek, we houden de code eenvoudig, maar maken wel iets superinteractief!

STAP 1 - MAPPEN EN BESTANDEN

We hebben een code-editor nodig. In de workshop gebruiken we Visual Studio Code.
  • 1.1 Maak een nieuwe map aan (vb op je bureaublad) en geef als naam soundboard
  • 1.2 Open het programma Visual Studio Code. en open de folder die je zopas aangemaakt hebt.
  • 1.3 Bouw onderstaande mappen- en bestandenstructuur aan
  • 1.5 Start de live server door rechtsonder de blauwe balk te klikken op Go Live
    Jouw standaard browser zal nu gestart worden en je zal een blanco webpagina zien!
  • GREAT SUCCESS

STAP 2 - HTML & CSS

Tijd om te coderen, zodat we meer te zien krijgen dan een witte webpagina!
  • 2.1 Kopieer onderstaande code in jouw index.html-bestand. Bewaar vervolgens dat bestand.
    Dit is een basis-structuur waarin we verder werken.
  • 2.2 Kopieer onderstaande code in jouw main.css-bestand. Bewaar vervolgens dat bestand.
    Dit bevat alvast een aantal basisstijlen die we later zullen nodig hebben voor de look & feel van onze app.
  • 2.3 Bekijk het resultaat in de browser (de url is hoogstwaarschijnlijk: http://127.0.0.1:5500/).
    Als alles goed is, ziet het er nu zo uit. Een volledig gekleurd vlak.
  • 2.4 Laten we het eens van kleur veranderen. Hiervoor moeten we de kleurwaarde veranderen. Zoek naar background-color: #07192f; en verander #07192f; naar bijvoorbeeld tomato; De achtergrond is nu tomaatkleurig.
  • 2.5 We hebben nu wat inhoud nodig voor onze webpagina.
    Voeg onderstaande twee elementen toe, en plaats ze tussen de openings <body id="background">-tag en sluitings </body>-tag
    <nav id="themenav"></nav>
    <div id="soundboard"></div>
    Alles is nu in gereedheid om het soundboard te initialiseren!

    Voeg deze regel code toe, onderaan net voor de sluitings-body-tag.

  • 3.3 Als het werkt, mag je de regel terug wissen, we weten nu dat ons JavaScript-bestand uitgevoerd wordt :)

STAP 4 - VOEG JE EIGEN SAMPLES TOE!

wetenschap is tof!
  • 4.1 Laten we een opname-bar toevoegen, dan kunnen we deel uitmaken van het soundboard!
    We zullen deze eerst structureel opbouwen, via het html-bestand.
    Voeg onderstaande code toe

    <div id="recordbar">
        <input id="name" type="text" class="form-control" placeholder="naam">
        <input id="title" type="text" class="form-control" placeholder="titel">
        <button id="record" class="btn btn-record">Start opname</button>
        <button id="upload" class="btn btn-upload">Upload geluid</button>
        <section id="player"></section>
        <p id="message"></p>
    </div>

    Deze ziet er zo uit:

  • 4.2 We laten deze nu werken door enerzijds alle nodige elementen te selecteren:
    (naam, titel, opnameknop, uploadknop, player, boodschap)
    en anderzijds functionaliteit toe te voegen aan elk element.
  • 4.3 Selecteren van elk element in app.js
  • 4.4 Initialiseer de opnamebar! Nu gaan we de microfoon-recorder inschakelen, en die laten werken als er op start geklikt wordt.
    Als men nadien klikt op de upload-knop, moet er een upload-functie in gang schieten, die we dadelijk aanvullen.
  • 4.5 En dan... tot slot... de inhoud van de uploadAction-functie

    Wat gebeurt er?

    • We geven de locatie waarnaar het mag geupload worden mee
    • We sturen de naam en de titel mee
    • Er wordt geupload met alle gegevens
    • We geven de boodschap weer die we terugkrijgen van de server
    • We resetten de recorder
    • We halen de nieuwe samples op!

  • GREAT SUCCESS
