STAP 1 - MAPPEN EN BESTANDEN

We hebben een code-editor nodig. In de workshop gebruiken we Visual Studio Code.
  • 1.1 Maak een nieuwe map aan (vb op je bureaublad) en geef als naam soundboard
  • 1.2 Open het programma Visual Studio Code. en open de folder die je zopas aangemaakt hebt.
  • 1.3 Bouw onderstaande mappen- en bestandenstructuur aan
  • 1.5 Start de live server door rechtsonder de blauwe balk te klikken op Go Live
    Jouw standaard browser zal nu gestart worden en je zal een blanco webpagina zien!
  • GREAT SUCCESS

STAP 2 - HTML & CSS

Tijd om te coderen, zodat we meer te zien krijgen dan een witte webpagina!
  • 2.1 Kopieer onderstaande code in jouw index.html-bestand. Bewaar vervolgens dat bestand.
    Dit is een basis-structuur waarin we verder werken.
  • 2.2 Kopieer onderstaande code in jouw main.css-bestand. Bewaar vervolgens dat bestand.
    Dit bevat alvast een aantal basisstijlen die we later zullen nodig hebben voor de look & feel van onze app.
  • 2.3 Bekijk het resultaat in de browser (de url is hoogstwaarschijnlijk: http://127.0.0.1:5500/).
    Als alles goed is, ziet het er nu zo uit. Een volledig gekleurd vlak.
  • 2.4 Laten we het eens van kleur veranderen. Hiervoor moeten we de kleurwaarde veranderen. Zoek naar background-color: #07192f; en verander #07192f; naar bijvoorbeeld tomato; De achtergrond is nu tomaatkleurig.
  • 2.5 We hebben nu wat inhoud nodig voor onze webpagina.
    Voeg onderstaande twee elementen toe, en plaats ze tussen de openings <body id="background">-tag en sluitings </body>-tag
    <nav id="themenav"></nav>
    <div id="soundboard"></div>
    Alles is nu in gereedheid om het soundboard te initialiseren!
